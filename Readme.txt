Use'make' command to compile and create executables for the 3 knapsack algorithms (bruteforce, backtracking and dynamic programming)
make creates 3 executables namely: kbt , kbf, kdp for backtrack, bruteforce and dynamic respectively.

To remove/cleanup the executale : type "make clean"

To run backtrack program: type 
./kbt 

To run bruteforce program: type
./kbf

To run dynamic program : type
./kdp
